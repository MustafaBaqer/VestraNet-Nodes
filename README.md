## 📌 Subscription Links (Auto-Updating)
Import these URLs directly into your client (v2rayNG, NekoBox, Shadowrocket, etc.).

| Type | Link | Format |
| :--- | :--- | :--- |
| **Full Mix (Recommended)** | `https://raw.githubusercontent.com/YourUser/VestraNet-Nodes/main/subscriptions/mix-base64.txt` | Base64 |
| **VLESS Only** | `https://raw.githubusercontent.com/YourUser/VestraNet-Nodes/main/protocols/vless.txt` | Normal |

## 🚀 How to Use
1. Copy the **Full Mix** URL.
2. Open your client app (e.g., v2rayNG).
3. Go to `Subscription Group Settings` -> `Add`.
4. Paste the URL and click `Update Subscription`.
