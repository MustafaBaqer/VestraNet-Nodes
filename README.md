## 🕊️ Internet Access is a Human Right
According to Article 19 of the Universal Declaration of Human Rights, everyone has the right to seek, receive, and impart information through any media and regardless of frontiers. 

The **VestraNet Nodes Ecosystem** is an advanced, automated proxy infrastructure designed to combat digital censorship, bypass Deep Packet Inspection (DPI), and dismantle authoritarian firewalls. We provide free, highly curated, and actively tested Xray/V2Ray and MTProto configurations to ensure absolute clarity and total control over your digital footprint.

## ⚙️ Core Engine Features
* **Real-Time Synchronization:** Configurations are dynamically evaluated and synced automatically by our core servers.
* **Anti-Censorship Protocols:** Native support for modern obfuscation techniques including **VLESS (Reality/XTLS)**, **VMESS**, **Trojan**, and **Shadowsocks**.
* **High-Performance Aggregation:** We curate nodes from top-tier open-source repositories, filtering out dead links to ensure maximum speed and low latency.
* **Telegram Unblocking:** Dedicated, sponsor-free MTProto proxies for seamless communication.

## 🚀 Compatible Clients & How to Use

Our configurations are compatible with industry-standard, open-source networking tools across all platforms.

### Recommended Apps:
* **Android:** [v2rayNG](https://github.com/2dust/v2rayNG) / [NekoBox for Android](https://github.com/MatsuriDayo/NekoBoxForAndroid)
* **Windows:** [v2rayN](https://github.com/2dust/v2rayN) / [NekoRay](https://github.com/MatsuriDayo/nekoray)
* **iOS / macOS:** [Shadowrocket](https://apps.apple.com/us/app/shadowrocket/id932747118) / [Streisand](https://apps.apple.com/us/app/streisand/id6450534064) / V2Box
* **Linux:** NekoBox / v2rayA

### Quick Setup Guide (Example for v2rayNG/v2rayN):
1. **Copy** the `Full Mix (Base64)` link from the table above.
2. **Open** your proxy application.
3. Navigate to **Subscription Group Settings** (or *Subgroup* / *Subscription*).
4. **Add** a new subscription, paste the copied URL, and save.
5. Click **Update Subscription** (or *Update via proxy*) to fetch the latest active nodes.
6. Select a node with low ping and connect.
