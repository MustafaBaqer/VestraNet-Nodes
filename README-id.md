## 🕊️ Akses Internet adalah Hak Asasi Manusia
Menurut Pasal 19 Deklarasi Universal Hak Asasi Manusia, setiap orang berhak atas kebebasan mempunyai dan mengeluarkan pendapat; hak ini mencakup kebebasan untuk mencari, menerima, dan menyampaikan informasi melalui media apa saja dan dengan tidak memandang batas-batas wilayah.

**Ekosistem Node VestraNet** adalah infrastruktur proksi otomatis canggih yang dirancang untuk memerangi sensor digital, melewati Deep Packet Inspection (DPI), dan membongkar firewall otoriter. Kami menyediakan konfigurasi Xray/V2Ray dan MTProto gratis, dikurasi secara ketat, dan diuji secara aktif untuk memastikan kejelasan mutlak dan kendali penuh atas jejak digital Anda.

## ⚙️ Fitur Inti Sistem
* **Sinkronisasi Waktu Nyata:** Konfigurasi dievaluasi secara dinamis dan disinkronkan secara otomatis oleh server inti kami.
* **Protokol Anti-Sensor:** Dukungan asli untuk teknik obfuskasi modern termasuk **VLESS (Reality/XTLS)**, **VMESS**, **Trojan**, dan **Shadowsocks**.
* **Agregasi Performa Tinggi:** Kami mengumpulkan node dari repositori sumber terbuka tingkat atas, menyaring tautan mati untuk memastikan kecepatan maksimum dan latensi (ping) rendah.
* **Buka Blokir Telegram:** Proksi MTProto khusus tanpa saluran sponsor untuk komunikasi yang lancar.

## 🚀 Klien yang Kompatibel & Cara Penggunaan

Konfigurasi kami kompatibel dengan alat jaringan standar industri dan sumber terbuka di semua platform.

### Aplikasi yang Direkomendasikan:
* **Android:** [v2rayNG](https://github.com/2dust/v2rayNG) / [NekoBox for Android](https://github.com/MatsuriDayo/NekoBoxForAndroid)
* **Windows:** [v2rayN](https://github.com/2dust/v2rayN) / [NekoRay](https://github.com/MatsuriDayo/nekoray)
* **iOS / macOS:** [Shadowrocket](https://apps.apple.com/us/app/shadowrocket/id932747118) / [Streisand](https://apps.apple.com/us/app/streisand/id6450534064) / V2Box
* **Linux:** NekoBox / v2rayA

### Panduan Penyiapan Cepat (Contoh untuk v2rayNG/v2rayN):
1. **Salin** tautan `Full Mix (Base64)` dari tabel di atas.
2. **Buka** aplikasi proksi Anda.
3. Navigasi ke **Subscription Group Settings** (Pengaturan Grup Langganan).
4. **Tambah** langganan baru, tempel URL yang disalin, dan simpan.
5. Klik **Update Subscription** (Perbarui Langganan) untuk mengambil node aktif terbaru.
6. Pilih node dengan ping rendah dan hubungkan.
