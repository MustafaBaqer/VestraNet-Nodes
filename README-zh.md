## 🕊️ 访问互联网是一项人权
根据《世界人权宣言》第19条，人人有权通过任何媒介寻求、接受和传递信息，不受国界限制。

**VestraNet 节点生态系统**是一个先进的、自动化的代理基础设施，旨在对抗数字审查、绕过深度包检测 (DPI) 并拆除专制防火墙。我们提供免费、高度精选且经过积极测试的 Xray/V2Ray 和 MTProto 配置，以确保对您的数字足迹拥有绝对的透明度和完全的掌控。

## ⚙️ 核心引擎特性
* **实时同步：** 配置由我们的核心服务器动态评估并自动同步。
* **反审查协议：** 原生支持现代混淆技术，包括 **VLESS (Reality/XTLS)**、**VMESS**、**Trojan** 和 **Shadowsocks**。
* **高性能聚合：** 我们从顶级的开源存储库中挑选节点，过滤掉失效链接，以确保最高的速度和低延迟。
* **Telegram 解锁：** 专用的、无赞助商广告的 MTProto 代理，确保顺畅通信。

## 🚀 兼容客户端及使用说明

我们的配置与所有平台上行业标准的开源网络工具兼容。

### 推荐应用：
* **Android:** [v2rayNG](https://github.com/2dust/v2rayNG) / [NekoBox for Android](https://github.com/MatsuriDayo/NekoBoxForAndroid)
* **Windows:** [v2rayN](https://github.com/2dust/v2rayN) / [NekoRay](https://github.com/MatsuriDayo/nekoray)
* **iOS / macOS:** [Shadowrocket](https://apps.apple.com/us/app/shadowrocket/id932747118) / [Streisand](https://apps.apple.com/us/app/streisand/id6450534064) / V2Box
* **Linux:** NekoBox / v2rayA

### 快速设置指南（以 v2rayNG/v2rayN 为例）：
1. **复制**上方表格中的 `Full Mix (Base64)` 链接。
2. **打开**您的代理应用程序。
3. 导航至**订阅分组设置**（或 *分组* / *订阅*）。
4. **添加**新订阅，粘贴复制的 URL 并保存。
5. 点击**更新订阅**（或 *通过代理更新*）以获取最新的活跃节点。
6. 选择一个延迟较低的节点并连接。
