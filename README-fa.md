# 🌐 اکوسیستم شبکه‌ VestraNet

![Update Status](https://img.shields.io/badge/آپدیت_خودکار-فعال-brightgreen?style=for-the-badge&logo=githubactions)

این مخزن شامل مجموعه‌ای از کانفیگ‌های قدرتمند، تست شده و پایدار (V2Ray/Xray و MTProto) است. سرورهای ما کیفیت و پینگ کانفیگ‌ها را بررسی کرده و این لیست را به صورت **ساعتی و کاملاً خودکار** آپدیت می‌کنند. شفافیت مطلق، کنترل کامل.

> 🌍 [Read in English](README.md)

## 📌 لینک‌های اشتراک (Subscription)
این لینک‌ها را کپی کرده و در برنامه‌هایی مانند **v2rayNG**، **NekoBox** یا **Shadowrocket** وارد کنید. کانفیگ‌های شما با یک کلیک آپدیت خواهند شد.

| نوع کانفیگ | لینک کپی (Raw) | فرمت |
| :--- | :--- | :--- |
| **ترکیبی کامل (پیشنهادی)** | `https://raw.githubusercontent.com/YourUser/VestraNet-Nodes/main/subscriptions/mix-base64.txt` | Base64 |
| **فقط VLESS** | `https://raw.githubusercontent.com/YourUser/VestraNet-Nodes/main/protocols/vless.txt` | خطی |
| **پروکسی تلگرام (MTProto)** | `https://raw.githubusercontent.com/YourUser/VestraNet-Nodes/main/protocols/mtproto.txt` | خطی |

## 🚀 راهنمای استفاده
۱. لینک **ترکیبی کامل** را از جدول بالا کپی کنید.
۲. وارد برنامه (مثلاً v2rayNG) شوید.
۳. از منوی کناری به بخش `Subscription Group Settings` بروید و یک گروه جدید بسازید.
۴. لینک را در بخش `URL` پیست (Paste) کنید.
۵. از منوی سه‌نقطه بالا، گزینه `Update Subscription` را بزنید تا جدیدترین کانفیگ‌ها دانلود شوند.
